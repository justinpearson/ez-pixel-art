When I twirl down the "image resize" panel, it appears to freeze because it takes so long to draw the lines for the resizer, because the initial width is the same width as the image, so there are tons of lines to draw. Instead, let's use some sort of outline AROUND the pixels. For simplicity, if its easier, it can just be a blue border around each pixel's square outline. That way we don't have to figure out how to "merge" the outlines around two adjacent pixels. Unless showing a "merged" outline is very easy, then we can just do it. This new way of presenting a set of selected pixels should apply not only to the Select tool, but in all other situations where a set of pixels is selected, eg, the "Select" button in the Color Quantization panel.
